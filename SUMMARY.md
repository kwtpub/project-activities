## Математическая основа

**Функция:**
```
f(x) = 1/(x² + 4x + 3) = 1/((x+1)(x+3))
```

**Полюсы:** x = -1, x = -3

**Первообразная (метод частичных дробей):**
```
F(x) = 1/2 · ln|(x+1)/(x+3)|
```

## Особенности реализации

### 1. Архитектура
- **Модульность**: Разделение на логические компоненты (функция, интегрирование, UI)
- **Статические методы**: Чистые функции без состояния
- **Обработка ошибок**: Использование исключений C++

### 2. Методы численного интегрирования

| Метод | Точность | Применение | Устойчивость |
|-------|----------|------------|--------------|
| Левые прямоугольники | O(h) | Задание 2 | Низкая |
| Средние точки | O(h²) | Задания 3, 5 | Высокая |
| Трапеции | O(h²) | Задание 4 | Низкая при особенностях |
| Главное значение Коши | Зависит от ε | Задание 5 | Специальный для особенностей |

### 3. Обработка особенностей

**Метод главного значения по Коши:**
```
P.V. ∫[a,b] f(x)dx = lim(ε→0) [∫[a,c-ε] f(x)dx + ∫[c+ε,b] f(x)dx]
```

где c - точка особенности, ε - малая окрестность исключения.
