# Численное интегрирование функции с особенностями

## Описание

Проект решает задачу численного интегрирования функции **f(x) = 1/(x² + 4x + 3)** с использованием различных методов:
- Метод левых прямоугольников
- Метод средних точек
- Метод трапеций
- Главное значение по Коши

## Функция и её свойства

```
f(x) = 1/(x² + 4x + 3) = 1/((x+1)(x+3))
```

**Особенности (полюсы):** x = -1, x = -3

**Первообразная:** F(x) = 1/2 · ln|(x+1)/(x+3)|

## Задания

1. **Задание 1:** Точное значение интеграла на A = [0, 1]
2. **Задание 2:** Левое правило (n = 7 узлов) на A = [0, 1]
3. **Задание 3:** Правило средних точек (n = 7) на A = [0, 1]
4. **Задание 4:** Особенности на B = [-1, 0], отсутствие сходимости (m = 10 итераций)
5. **Задание 5:** Главное значение по Коши на C = [-2, 0]

## Результаты

### Интервал A = [0, 1] (без особенностей)

- **Точное значение:** -0.0810930216
- **Левое правило (n=7):** погрешность O(h)
- **Средние точки (n=7):** погрешность O(h²)

### Интервал B = [-1, 0] (особенность x = -1)

- Метод трапеций **не сходится** из-за полюса на границе
- Демонстрируется нестабильность результатов

### Интервал C = [-2, 0] (особенность x = -1 внутри)

- Главное значение по Коши: **-0.6931471806** ≈ -ln(2)
- Сходимость при уменьшении epsilon
